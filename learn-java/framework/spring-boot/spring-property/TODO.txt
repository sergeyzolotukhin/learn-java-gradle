spring.config.additional-location
- The Spring Boot property spring.config.additional-location is used to specify one or more locations
where Spring Boot should search for configuration files, such as application.properties or application.yml,
in addition to the default locations.

spring.config.location
- Once spring.config.name is set externally,
Spring Boot will then look for files with that base name (e.g., myconfig.properties, myconfig.yml)
in its default search locations, or in locations specified by spring.config.location.
