Ansible-Vagrant-ELK
=====================

This is an Ansible playbook to setup ELK environment

Environment
-----------------
1. Run Vagrant

   ``` bash
      vagrant up
   ```

2. Open [Kibana](http://192.168.2.210:5601/app/kibana) in browser
